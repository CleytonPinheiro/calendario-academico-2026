📅 Painel de Planejamento 2026 - SEED PR Digital

Uma ferramenta interativa e dinâmica desenvolvida para auxiliar professores da rede estadual do Paraná (SEED-PR) na organização de seus registros de aula e planejamento trimestral para o ano letivo de 2026.

🚀 Funcionalidades

Calendário Dinâmico: Navegação entre os meses de 2026 com atualização automática de feriados e recessos oficiais.

Campos de Preenchimento Claros: Áreas designadas para o nome do professor e disciplina em cada bloco de aula.

Seleção de Período: Marcação rápida da 1ª à 6ª aula do dia através de círculos interativos.

Controle de Avaliações: Checkboxes específicas para acompanhamento de AV40, AV60 e Recuperação (REC).

Bloqueio de Dias: Possibilidade de clicar em qualquer dia para alternar o status entre "Letivo" e "Não Letivo" (bloqueio visual).

Otimização para Impressão: Layout configurado para folha A4 (Retrato), ocultando botões de navegação e menus durante a impressão.

Design Adaptativo: Interface moderna construída com Tailwind CSS, focada em usabilidade.

🛠️ Tecnologias Utilizadas

HTML5: Estrutura semântica.

Tailwind CSS: Estilização responsiva e utilitária.

JavaScript (Vanilla): Lógica do calendário, manipulação de datas e estados de bloqueio.

Google Fonts (Inter): Tipografia limpa para facilitar a leitura.

📦 Como usar

Clone o repositório:

git clone [https://github.com/seu-usuario/seu-repositorio.git](https://github.com/seu-usuario/seu-repositorio.git)


Abra o arquivo:
Localize o arquivo 2026_Interactive_Calendar_Dashboard.html e abra-o em qualquer navegador moderno.

Planeje e Imprima:

Navegue até o mês desejado.

Bloqueie/Desbloqueie dias conforme seu cronograma escolar.

Clique em IMPRIMIR A4 para gerar a versão física do seu planejamento.
